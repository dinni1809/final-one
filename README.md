# FAATTSOO Mobile

FAATTSOO is a mobile-first Expo React Native restaurant discovery application built from the uploaded screenshots.

## Run

```bash
npm install
npm run start
```

Set `EXPO_PUBLIC_API_URL` to connect the existing backend:

```bash
EXPO_PUBLIC_API_URL=http://localhost:5000/api
```

The app includes mock fallbacks for restaurants, offers, ratings, and menu data so the UI remains usable while the backend is offline.

## Stack

- Expo
- React Native
- TypeScript
- React Navigation
- React Query
- Zustand
- Axios
- React Hook Form
- Zod
- Expo Secure Store
- Expo Image
- Expo AV
